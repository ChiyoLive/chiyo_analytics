Duration Tracking

The SDK tracks active stay duration accurately. If you spend less than 1 second on a page, it is ignored to keep metrics clean.

🧭

Multi-Page Routing (MPA)

Unlike SPAs, routing here uses native browser page reloads. The SDK automatically detects page load and reports stay durations when navigating away.

🗃️

Session Durability

Visitor and Session identifiers are automatically stored inside `localStorage` and `sessionStorage` to trace persistent journeys.

How it works

When loading any HTML page, the SDK script is injected in the header. It automatically reads site parameters and starts the session tracker: 

<script src="sdk.js"
        data-collector-url="http://localhost:8080/collect"
        data-geo-lookup-url="http://localhost:8080/collect/geo"
        data-site-id="example-traditional-mpa"
        data-token-url="/api/cyanly-token"></script>

This sends pageview duration beacons and UTM parameters when you navigate away or close the tab. You can inspect the Live Ingest Console below to see events recorded across pages!
